# AI Powered Gig Worker Insurance (Full Prototype)

## Project Overview
This project is an AI-powered insurance platform designed for gig workers such as drivers, delivery partners, and freelancers.  
It helps workers register, view insurance plans, and submit claims easily.

## Features
- Worker registration
- Insurance policy management
- Claims submission
- AI risk analysis (prototype)
- Simple dashboard interface

## Tech Stack
Frontend: HTML, CSS, JavaScript  
Backend: Python (Flask)  
Database: JSON / Local storage

## Installation

Clone the repository

git clone https://github.com/bindusree1209/gig-worker-insurance.git

Navigate to the project folder

cd gig-worker-insurance

## Run Backend

cd backend  
pip install -r requirements.txt  
python app.py  

Server runs at:  
http://127.0.0.1:5000

## Run Frontend

Open the following file in your browser:

frontend/index.html
