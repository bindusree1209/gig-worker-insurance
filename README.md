# AI Powered Gig Worker Insurance (Full Prototype)

## Project Overview

This project is a prototype platform designed to provide insurance services for gig workers such as drivers, delivery partners, and freelancers.
The system allows workers to register, explore insurance options, and submit claims through a simple interface.

## Features

* Worker registration
* Insurance policy management
* Claim submission system
* AI-based risk analysis (prototype)
* Simple user interface

## Tech Stack

Frontend: HTML, CSS, JavaScript
Backend: Python (Flask)
Database: Local storage / JSON

## Installation

Clone the repository

git clone https://github.com/bindusree1209/gig-worker-insurance.git

Navigate to the project folder

cd gig-worker-insurance

## Run Backend

cd backend
pip install -r requirements.txt
python app.py

Server runs at:
http://localhost:5000

## Run Frontend

Open the following file in your browser:

frontend/index.html
